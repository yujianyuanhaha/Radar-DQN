mdp_span

Computes the span of a vector.

Syntax

sp = span(W)

Description

mdp_span computes the span of the W vector :
     max W(s) - min W(s)
       s                  s

Argument

  • W : a vector.

Evaluation

  • sp : the value of the span of the W vector.

Example
>> W = [1; 2; 3; 4];

>> sp = mdp_span(W)
sp =
   3
